# 《主播模拟器：双面人生》剧情策划方案 V4 — 最终版

> **核心理念**：翻车即内容，失控即流量，社死即封神。
> 
> **精神内核**：没有说教，只有讽刺 — 社会批判藏在笑声里，不是按着玩家的头说"看，这就是现实"。

## 一、核心概念

### 1.1 小爱是谁？

小爱是**千千万万普通青年的缩影** —— 喜欢在网络上分享生活，给大家带来快乐，实际生活中腼腆、不善交谈，但善良温暖。

**基础设定：**
- **姓名**：林小爱（主播名：小爱）
- **年龄**：22岁
- **身份**：刚毕业大学生，找不到工作，被迫当主播还债
- **性格**：镜头前开朗活泼、风趣幽默；镜头后腼腆内向、不善交际

**双重身份设定：**
- **镜头前（虚拟身份）**：闪闪发光的小太阳，被人喜欢、充满自信
- **镜头后（现实身份）**：家境普通、生活窘迫、被家人否定的底层年轻人

**核心矛盾**：
> 现实里的不被认可，让人们依赖网络寻找归属感；网络上的光鲜人设，又让人们渐渐逃避真实的自己。

### 1.2 核心循环

```
开始直播 → AI弹幕互动（8人格）→ 触发翻车/随机事件 → 做出选择 → 
数值变化 → 可能进入"失控模式" → 下播 → 现实事件入侵 → 
热搜生成 → 下一天 → 20天后触发停播100天 → 6种真相版本 → 
回归直播 → AI生涯总结报告
```

| 状态 | 范围 | 效果 |
|------|------|------|
| 😇 正常 | 80-100 | 直播正常，弹幕觉得"太无聊"，人气涨幅减半 |

## 四、弹幕人格系统

### 4.1 8个常驻粉丝人格

这些不是好感度NPC，而是会记住你一切并在弹幕里反复鞭尸的"互联网活化石"。

| ID | 人格名 | 特征 | 典型弹幕 |
|----|--------|------|----------|
| 001 | **老粉阿伟** | 从第一天就在看，专门翻旧账 | "我记得你上次也是这么说的"、"主播又双叒叕翻车了" |
| 002 | **黑粉头子** | 专门挑刺但打赏最多（氪金黑粉） | "唱得什么玩意儿，但我就爱看这个"（附赠火箭） |
| 003 | **翻译君** | 把主播的话翻译成奇怪语言 | 主播："谢谢大哥" → 翻译："谢主隆恩，吾皇万岁万岁万万岁" |
| 004 | **颜文字怪** | 只发颜文字和emoji | "(๑°o°๑) (☆▽☆) (；′⌒`)" |
| 005 | **考据党** | 统计主播的一切数据 | "今日翻车次数：3次，累计翻车：47次，超越全国99%主播" |
| 006 | **奶奶粉** | 像奶奶一样关心主播，但经常误解梗 | "主播怎么哭了？是不是有人欺负你？奶奶帮你骂他" |
| 007 | **梗百科** | 行走的互联网梗词典 | "这波啊，这波是经典复刻"、"主播在第五层" |
| 008 | **潜水员** | 平时不说话，关键时刻神评论 | （平时 silence）→ "主播，你麦克风没关" |

### 4.2 AI生成逻辑

在弹幕生成prompt中加入：

```
弹幕中必须包含以下常驻粉丝的发言（每人1-2条）：
{personality_prompts}

注意：
- 这些粉丝记得主播之前发生的事（{recent_history}）
- 老粉阿伟可能提到主播上次翻车的事
- 黑粉头子可能在骂人的同时打赏
- 翻译君可能把主播的话翻译成文言文/东北话
- 考据党会统计主播的翻车频率、涨粉速度
- 奶奶粉会用温暖的语气误解主播说的话
```

### 4.3 记忆机制

用**滚动摘要**维护"最近5次重大事件"：

```typescript
interface RecentHistory {
  last5Events: string[];     // 最近5个重大事件
  last5Fails: string[];      // 最近5次翻车
  runningGags: string[];     // 连续梗（如"主播已连续3天迟到"）
}
```

## 六、停播100天真相系统（保留同学核心剧情）

### 6.1 核心设定

**元叙事结构**：
- 玩家作为小爱的忠实粉丝，在停播第100天收到直播推送
- 观看直播时突然穿越，绑定角色林小爱，回到起步阶段
- 机械音提示："请走完小爱的人生之路，找到她停播100天的真相"
- 玩家的每一个选择，都在改写小爱的人生，也在揭示停播的真相

### 6.2 停播真相的揭示

**真相只有一个**：
> 小爱并非遭遇意外，而是在两种身份（虚拟vs现实）的挣扎中陷入精神内耗，选择暂停直播，寻找真实自我。

**揭示方式**：
- 根据玩家20天的选择，AI生成个性化的"真相版本"
- 不同选择导向不同的"精神内耗形态"，但核心都是身份认同危机

| 玩家选择倾向 | 停播真相形态 | 揭示台词 |
|-------------|-------------|---------|
| 一直维持完美人设 | **表演型人格崩溃** | "我演得太累了，忘了自己是谁" |
| 频繁翻车/失控 | **精神崩溃疗养** | "我需要停下来，找回理智" |

## 九、多结局系统（5个核心结局）

基于同学的原始结局，保留核心5结局：

### 结局一：自我和解·回归本真（偏圆满）

**触发条件**：
- 善良值≥60，诚信值≥60
- 妈妈好感度≥60，豆豆好感度≥60，可心好感度≥60（或已锁定）
- 人设完整度>50

**剧情**：
主角暂停直播，回到老家陪伴妈妈，关掉虚拟形象，分享真实生活。粉丝没有离开，反而更加喜欢真实的主角。妈妈病情逐渐稳定，豆豆陪伴在身边，月牙儿成为朋友，偶尔联动直播。主角终于明白，互联网身份只是一部分，真正的身份认同，是接纳现实中的自己，兼顾事业与亲情，实现了虚拟与现实的和解。

### 结局二：困于虚拟·彻底迷失（偏悲剧）

**触发条件**：
- 善良值≤40，诚信值≤40
- 妈妈好感度≤30，可心好感度≤30，豆豆好感度≤30
- 人设完整度<30

**剧情**：
主角坚守互联网事业，成为顶级大博主，拥有巨额财富和千万粉丝，但失去了所有现实中的牵挂——妈妈彻底失忆，不认识主角；可心、豆豆早已淡出生活；月牙儿成为敌对关系。主角深夜坐在豪华公寓里，倍感孤独，最终选择全网断联，躲起来独自承受遗憾，活成了互联网的傀儡，弄丢了真正的自己。

### 结局三：逃离网络·回归现实（中性）

**触发条件**：
- 精神值<20，主动选择退网
- 遭遇严重网暴

**剧情**：
主角终于意识到，网络上的光鲜都是虚无缥缈的，现实的温暖与安稳才是自己真正想要的。停播100天里，彻底告别主播身份，删除所有直播账号，回到老家，找一份稳定的工作，陪伴在家人身边，彻底抛弃虚拟身份，回归平淡却真实的现实生活。

### 结局四：双向崩塌·彻底沉寂（失败结局）

**触发条件**：
- 经济值≤-1000
- 房东太太好感度=0
- 善良值<20

**剧情**：
主角被新房东赶出出租屋，经济彻底破产，直播设备被变卖，人气值骤降至0。所有人物均已淡出生活，主角无法继续直播，也无力回家面对妈妈，最终放弃挣扎，重新沦为无人问津的普通人，带着满身疲惫，在迷茫中继续挣扎。

### 结局五：真相大白·双向救赎（完美结局）

**触发条件**：
- 善良值≥70，诚信值≥70
- 所有人物好感度≥60
- 解锁停播100天"主动修行"版本

**剧情**：
完美剧情线：妥善处理所有矛盾，既坚守了善良本心，也坚持了直播理想，用真诚收获现实的认可与网络的喜爱。停播100天的真相揭晓：你并非遭遇意外，而是在两种身份的挣扎中陷入精神内耗，选择暂停直播，寻找真实自我。最终带着粉丝的理解、现实的温暖，重新回归，用真实的自我直播，实现虚拟与现实的双向救赎，成为真正被认可的自己。

## 十二、总结：为什么这个设计能赢

1. **AI是核心玩法，不是装饰品** — 每次游玩内容都不同，重玩价值极高
2. **翻车=内容=流量** — 完美讽刺真实直播生态，玩家会心一笑
3. **弹幕人格让世界鲜活** — 8个有记忆的"粉丝"比6个好感度NPC更有趣
4. **没有说教，只有讽刺** — 社会批判藏在笑声里
5. **停播100天是高潮不是终点** — 6种真相版本给玩家惊喜
6. **保留同学的核心剧情** — 6个节点、5个结局、人物关系完整保留
7. **与现有系统完美融合** — 天赋树、四分区、数值系统都不冲突
8. **终局有记忆点** — AI生成的生涯总结是完美的"分享截图"素材
