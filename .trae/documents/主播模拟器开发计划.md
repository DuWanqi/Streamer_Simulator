# 主播模拟器：双面人生 - 开发计划

## 项目概述

基于现有TypeScript + Vite + PixiJS项目，实现完整版剧情策划方案，打造专业二次元风格Web端游戏。

***

## 一、项目架构设计

* **低耦合**：各模块独立，通过事件系统通信

* **可扩展**：易于添加新事件、新场景

* **类型安全**：完整TypeScript类型定义

* **UI专业**：二次元风格，无AI味

**双轨制**:

* **30个随机翻车事件**: 30%概率触发

**事件触发逻辑**:

```typescript
class EventManager {
  checkDailyEvent(day: number): GameEvent | null {
    // 1. 检查是否有核心剧情节点
    const storyNode = this.getStoryNode(day);
    if (storyNode) return storyNode;
    
    // 2. 30%概率触发随机事件
    if (Math.random() < 0.3) {
      return this.getRandomEvent();
    }
    
    return null;
  }
}
```

## 三、关键技术实现

### 3.1 图片背景处理

**方案**: Canvas像素操作

```typescript
export class ImageProcessor {
  static async removeWhiteBackground(imageUrl: string): Promise<string> {
    const img = await loadImage(imageUrl);
    const canvas = document.createElement('canvas');
    const ctx = canvas.getContext('2d')!;
    
    canvas.width = img.width;
    canvas.height = img.height;
    ctx.drawImage(img, 0, 0);
    
    const imageData = ctx.getImageData(0, 0, canvas.width, canvas.height);
    const data = imageData.data;
    
    // 遍历像素，将白色背景设为透明
    for (let i = 0; i < data.length; i += 4) {
      const r = data[i];
      const g = data[i + 1];
      const b = data[i + 2];
      
      // 检测白色背景（容差）
      if (r > 240 && g > 240 && b > 240) {
        data[i + 3] = 0; // 设置透明
      }
    }
    
    ctx.putImageData(imageData, 0, 0);
    return canvas.toDataURL('image/png');
  }
}
```

### 3.2 状态管理

**使用观察者模式**:

```typescript
export class GameState extends EventTarget {
  private state: PlayerState;
  
  setState<K extends keyof PlayerState>(
    key: K, 
    value: PlayerState[K]
  ) {
    const oldValue = this.state[key];
    this.state[key] = value;
    
    this.dispatchEvent(new CustomEvent('stateChange', {
      detail: { key, oldValue, newValue: value }
    }));
  }
  
  getState(): Readonly<PlayerState> {
    return { ...this.state };
  }
}
```

### 3.3 场景切换动画

```typescript
export class SceneTransition {
  static async fadeOut(element: HTMLElement): Promise<void> {
    return new Promise(resolve => {
      element.style.transition = 'opacity 0.3s';
      element.style.opacity = '0';
      setTimeout(resolve, 300);
    });
  }
  
  static async fadeIn(element: HTMLElement): Promise<void> {
    return new Promise(resolve => {
      element.style.opacity = '0';
      element.style.transition = 'opacity 0.3s';
      requestAnimationFrame(() => {
        element.style.opacity = '1';
        setTimeout(resolve, 300);
      });
    });
  }
}
```
