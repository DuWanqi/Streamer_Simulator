# 《主播模拟器：双面人生》剧情策划方案 V3 — 最终整合版

> **核心理念**：翻车即内容，失控即流量，社死即封神。
> 
> **精神内核**：没有说教，只有讽刺 — 社会批判藏在笑声里，不是按着玩家的头说"看，这就是现实"。

---

## 一、核心概念重构

### 1.1 小爱是谁？

小爱是一个**运气极差但生命力极强的主播**，被迫在虚拟与现实之间走钢丝。

**基础设定：**
- **姓名**：小爱（AI，暗示她是"被算法选中的人"）
- **年龄**：22岁
- **身份**：刚毕业大学生，找不到工作，被迫当主播还债
- **口头禅**："家人们谁懂啊"、"这破直播是一天也播不下去了"、"谢谢大哥的火箭"

**双重身份设定：**
- **镜头前**：努力维持人设的"超天酱"——可爱、努力、正能量
- **镜头后**：被现实毒打的普通女孩——交不起房租、被家人不理解、随时可能崩溃

**关键机制**：
- 玩家需要同时管理"虚拟人设"和"现实生活"
- 两个人格会互相影响：人设越完美，现实越崩塌；现实问题越多，直播越容易失控
- 没有"好结局"或"坏结局"，只有"不同风格的社死与封神"

### 1.2 核心循环

```
开始直播 → 选择流派 → AI弹幕互动 → 触发翻车/随机事件 → 做出选择 → 
数值变化 → 可能进入"超天酱模式"或"崩溃模式" → 下播 → 
现实事件入侵 → 处理生活问题 → 热搜生成 → 下一天
     ↑                                                                      
     └──────── 翻车内容成为下次直播的话题素材 ←──────────────────────────────┘
```

## 二、直播分区系统（保留现有四区）

| 分区 | 名称 | 特色 | 翻车风格 |
|------|------|------|----------|
| 🎵 | **音乐歌手区** | 唱歌、弹唱、音乐创作 | 破音、忘词、设备故障、美颜失效 |
| 💃 | **舞蹈区** | 宅舞、街舞、形体训练 | 动作失误、服装事故、体力不支、背景乱入 |
| 🎮 | **游戏区** | 各类游戏直播 | 操作下饭、被虐哭、设备崩溃、口吐芬芳 |
| 🤡 | **整活搞笑区** | 抽象、挑战、日常分享 | 本身就是翻车，翻车就是内容 |

**分区特色机制：**
- 不同分区触发不同类型的翻车事件
- 分区选择影响弹幕人格的说话风格
- 某些事件只在特定分区触发（如"游戏连败"只在游戏区）

## 三、精神状态仪表盘（五维数值系统）

玩家需要同时管理5个核心数值，所有数值可视化显示（进度条+emoji）：

| 数值 | 图标 | 作用 | 过高后果 | 过低后果 |
|------|------|------|----------|----------|
| **粉丝数** | 👥 | 基础目标，要涨粉 | 黑粉比例增加 | 游戏失败 |
| **精神值** | 🧠 | 主角的理智程度 | 触发"超天酱"模式 | 抑郁/崩溃结局 |
| **道德底线** | ⚖️ | 还能不能算个人 | 解锁无下限操作 | 圣母/被欺负结局 |
| **钱包厚度** | 💰 | 能不能活下去 | 被MCN绑架 | 饿死/退网 |
| **人设完整度** | 🎭 | 虚拟vs真实的撕裂感 | 彻底迷失 | 暴露真实自我 |

**数值联动机制：**
- 任何操作都会影响多个数值（trade-off设计）
- "精神值↓"可以换来"粉丝数↑"，但可能触发崩溃
- "道德底线↓"可以短期涨粉，但长期可能被封杀

### 3.1 "超天酱"模式

当**精神值>80**时，小爱进入"超天酱"状态：
- 直播效果爆炸，所有收益×3
- 解锁疯狂操作：直播睡觉、发呆、骂粉丝（反向涨粉）
- 弹幕变成"主播疯了但好可爱"
- 但精神值会快速下降，降到0触发"崩溃直播"

**崩溃直播：**
- 小爱在直播间大哭/大笑/胡言乱语
- 粉丝数暴涨（吃瓜观众涌入）
- 第二天触发"网暴事件"或"热搜霸榜"

### 3.2 精神状态等级

| 状态 | 范围 | 效果 |
|------|------|------|
| 😇 正常 | 80-100 | 直播正常，但弹幕觉得"太无聊"，人气涨幅减半 |
| 😅 紧张 | 60-79 | 偶尔说错话，弹幕觉得"有点意思"，人气正常 |
| 🤪 上头 | 40-59 | 放飞自我，直播离谱，人气暴涨但不可控 |
| 😵 失控 | 20-39 | 完全不按剧本，AI弹幕疯狂，热搜频出 |
| 💀 崩溃 | 0-19 | 直播间变人间奇观，全网围观，解锁传说内容 |

**核心悖论：正常直播没人看，失控了反而爆火。**

## 四、直播流派系统（Roguelike Build）

每场直播可以选择一个"流派"，不同流派影响数值变化：

| 流派 | 核心机制 | 适合玩家 |
|------|----------|----------|
| **炒作狗** | 道德底线↓=粉丝数↑↑ | 喜欢无下限的玩家 |
| **实力派** | 才艺↑=稳定涨粉 | 稳健型玩家 |
| **抽象派** | 随机事件概率↑ | 喜欢刺激的玩家 |
| **卖惨流** | 精神值↓=同情粉↑ | 情感操控大师 |
| **技术宅** | 知识↑=高质量粉↑ | 硬核玩家 |
| **颜值党** | 外貌↑=土豪打赏↑ | 简单直接 |

**Build组合示例：**
- "疯批美人"：炒作狗+卖惨流+超天酱模式
- "知识抽象家"：技术宅+抽象派
- "纯欲天花板"：颜值党+实力派

## 五、弹幕人格系统

### 5.1 8个常驻粉丝人格

这些不是好感度NPC，而是会记住你一切并在弹幕里反复鞭尸的"互联网活化石"。

| ID | 人格名 | 特征 | 典型弹幕 |
|----|--------|------|----------|
| 001 | **老粉阿伟** | 从第一天就在看，专门翻旧账 | "我记得你上次也是这么说的"、"主播又双叒叕翻车了" |
| 002 | **黑粉头子** | 专门挑刺但打赏最多（氪金黑粉） | "唱得什么玩意儿，但我就爱看这个"（附赠火箭） |
| 003 | **翻译君** | 把主播的话翻译成奇怪语言 | 主播："谢谢大哥" → 翻译："谢主隆恩，吾皇万岁万岁万万岁" |
| 004 | **颜文字怪** | 只发颜文字和emoji | "(๑°o°๑) (☆▽☆) (；′⌒`)" |
| 005 | **考据党** | 统计主播的一切数据 | "今日翻车次数：3次，累计翻车：47次，超越全国99%主播" |
| 006 | **奶奶粉** | 像奶奶一样关心主播，但经常误解梗 | "主播怎么哭了？是不是有人欺负你？奶奶帮你骂他" |
| 007 | **梗百科** | 行走的互联网梗词典 | "这波啊，这波是经典复刻"、"主播在第五层" |
| 008 | **潜水员** | 平时不说话，关键时刻神评论 | （平时 silence）→ "主播，你麦克风没关" |

### 5.2 AI生成逻辑

在弹幕生成prompt中加入：

```
弹幕中必须包含以下常驻粉丝的发言（每人1-2条）：
{personality_prompts}

注意：
- 这些粉丝记得主播之前发生的事（{recent_history}）
- 老粉阿伟可能提到主播上次翻车的事
- 黑粉头子可能在骂人的同时打赏
- 翻译君可能把主播的话翻译成文言文/东北话
```

### 5.3 记忆机制

用**滚动摘要**维护"最近5次重大事件"：

```typescript
interface RecentHistory {
  last5Events: string[];     // 最近5个重大事件
  last5Fails: string[];      // 最近5次翻车
  runningGags: string[];     // 连续梗（如"主播已连续3天迟到"）
}
```

## 六、翻车事件池（30个核心事件）

每场直播后会触发1-2个随机事件，玩家需要做出选择。

### 6.1 技术翻车类（6个）

**事件1：美颜失效**
- 描述：你精心设置的美颜滤镜突然失效，前后对比被截图
- 选项：
  1. 假装无事发生（人设完整度-10，粉丝数+1000，弹幕截图狂欢）
  2. 大方自嘲（人设完整度+5，粉丝数+3000，"真实主播"标签）
  3. 紧急下播修设备（粉丝数-1000，精神值-10）
  4. 把素颜当卖点（抽象派+1，粉丝数+5000，解锁"真实美"线）

**事件2：忘关麦/忘关摄像头**
- 描述：下播后忘关麦，你骂粉丝/吐槽平台的话被全听到
- 选项：
  1. 假装被盗号（粉丝数-5000，弹幕："主播当我们傻？"）
  2. 承认并道歉（粉丝数-2000，精神值-10）
  3. 立"真性情"人设（道德底线-10，粉丝数+3000，黑粉变铁粉）
  4. 继续骂，骂得更狠（抽象派+2，粉丝数+10000，精神值-20，热搜预定）

**事件3：直播软件崩溃**
- 描述：软件崩溃重启，摄像头切到后置镜头，出租屋全貌曝光
- 选项：
  1. 紧急切换（粉丝数-1000）
  2. 直播room tour（抽象派+1，粉丝数+3000，"比我还破，安心了"）
  3. 假装是故意策划（道德底线-5，粉丝数+2000）

**事件4：弹窗入侵**
- 描述：Windows更新/杀毒软件/奇怪广告弹窗挡住脸
- 选项：
  1. 关掉继续（无事发生）
  2. 和弹窗互动（抽象派+1，粉丝数+2000）
  3. 直播卸载软件（粉丝数+1000，可能触发软件商投诉）

**事件5：设备连环故障**
- 描述：麦克风没声→换耳机→耳机漏电→摄像头花屏
- 选项：
  1. 坚持直播（精神值-20，粉丝数+5000，"敬业主播"）
  2. 下播维修（粉丝数-2000）
  3. 用手机继续（抽象派+1，画质渣但真实）

**事件6：AI换脸入侵**
- 描述：有人用你的脸做了AI换脸视频，内容是...
- 选项：
  1. 律师函警告（无事发生）
  2. 直播看换脸视频（抽象派+2，精神值-20，粉丝数+10000）
  3. 自己也做换脸反击（道德底线-30，进入赛博朋克线）
  4. 假装是自己拍的（道德底线-50，粉丝数+20000）

## 七、停播100天真相系统（保留核心剧情）

### 7.1 触发条件

当玩家完成20天直播后，根据累计数值和关键选择，触发"停播100天"事件。

**不是惩罚，而是必然发生的剧情节点** —— 就像主播总会遇到瓶颈期。

### 7.2 停播真相的6种版本

根据前期选择，100天的真相完全不同：

| 版本 | 触发条件 | 真相内容 | 回归后变化 |
|------|---------|---------|-----------|
| **精神疗养** | 精神值累计<30 | 小爱真的崩溃了，被送进精神病院治疗 | 解锁"疯批美人"模式，精神值上限+20 |
| **MCN囚禁** | 签约MCN且违约 | 被MCN软禁，被迫每天播16小时 | 解锁"反MCN斗士"线，道德底线-30但粉丝粘性+50% |
| **算法觉醒** | 接受算法次数>3 | 小爱发现自己是AI实验品，停播是系统维护 | 解锁"赛博主播"线，可以操控AI观众 |
| **现实崩塌** | 人设完整度<20 | 现实身份被曝光，被全网人肉，躲了100天 | 解锁"真实自我"线，不再维护人设 |
| **主动修行** | 道德底线>80且精神值>80 | 小爱主动退网100天寻找自我 | 解锁"佛系主播"线，涨粉慢但稳定 |
| **平行宇宙** | 触发"穿越"事件 | 小爱真的穿越到了平行世界 | 解锁"二周目"模式，保留所有记忆 |

### 7.3 100天期间的事件

停播期间，玩家每天会收到一条"外界消息"（文字+AI生成）：
- 粉丝在找你
- 黑粉在庆祝
- 房东在催租
- 室友在用你的账号
- 平台算法在"想念"你

**玩家可以选择：**
- 偷偷看但不回应（精神值+5）
- 完全断网（精神值+10，但粉丝流失）
- 用小号观察（解锁"小号"线）

### 7.4 回归直播

100天后，小爱回归直播，AI生成一场"回归首秀"：
- 弹幕人格会提到"你终于回来了"
- 黑粉头子会说"还以为你死了"
- 考据党会统计"停播期间粉丝流失数据"

**回归选择：**
1. 解释真相（根据版本不同，粉丝反应不同）
2. 不解释，直接开播（抽象派+3，"谜语人"标签）
3. 编个故事（道德底线-10，但可能更有趣）

## 十三、总结：为什么这个设计能赢

1. **AI是核心玩法，不是装饰品** — 每次游玩内容都不同，重玩价值极高
2. **翻车=内容=流量** — 完美讽刺真实直播生态，玩家会心一笑
3. **弹幕人格让世界鲜活** — 8个有记忆的"粉丝"比6个好感度NPC更有趣
4. **没有说教，只有讽刺** — 社会批判藏在笑声里
5. **停播100天是高潮不是终点** — 给玩家一个"中场休息"和"剧情转折"
6. **系统灵活可扩展** — 新增一个翻车模板比写一段分支剧情简单100倍
7. **终局有记忆点** — AI生成的生涯总结是完美的"分享截图"素材
