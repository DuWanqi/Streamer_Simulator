# 《主播模拟器：双面人生》剧情策划方案 - 最终整合版

> **核心理念**：翻车即内容，失控即流量，社死即封神。
> 
> **精神内核**：没有说教，只有讽刺 — 社会批判藏在笑声里。

---

## 一、核心设定

### 1.1 故事框架（元叙事）

**开篇**：
- 玩家是小爱的忠实粉丝，停播100天后收到直播推送
- 观看时突然穿越，绑定角色林小爱，回到起步阶段（第1天）
- 机械音："请走完小爱的人生之路，找到她停播100天的真相"

**真相**（20天后揭示）：
> 小爱并非遭遇意外，而是在虚拟与现实两种身份的挣扎中陷入精神内耗，选择暂停直播，寻找真实自我。

### 1.2 小爱是谁

- **姓名**：林小爱（主播名：小爱）
- **年龄**：22岁
- **身份**：刚毕业大学生，找不到工作，被迫当主播还债
- **双重身份**：
  - 镜头前：闪闪发光的小太阳，被人喜欢、充满自信
  - 镜头后：家境普通、生活窘迫、被家人否定的底层年轻人

## 二、游戏流程（20天结构）

### 整体时间线

| 天数 | 阶段 | 核心内容 | 出现NPC |
|------|------|----------|---------|
| 第1天 | 开篇 | 玩家穿越，初始探索 | 房东太太（首次） |
| 第2-5天 | 初期生存 | 房租危机，努力涨粉 | 可心（第3天出现） |
| 第6-10天 | 涨粉阶段 | 捡到豆豆，粉丝增长 | 豆豆（第6天出现） |
| 第11-15天 | 网暴危机 | 猥琐男骚扰，应对网暴 | 猥琐男（第11天出现） |
| 第16-19天 | 巅峰与迷失 | 与月牙儿PK，成为大主播 | 月牙儿（第16天出现） |
| 第20天 | 最终抉择 | 母亲阿尔茨海默症 | 妈妈（电话/回忆） |
| 结局 | 真相揭示 | 停播100天真相 + 5种结局 | 所有NPC |

### 每日流程

```
每天固定流程：
1. 白天：现实事件（可能触发NPC互动）
2. 晚上：开启直播
   - AI弹幕互动（8人格系统）
   - 可能触发：随机翻车事件 或 核心剧情节点
   - 玩家做出选择
   - 数值变化
3. 下播：热搜生成（如有重大事件）
4. 结算：查看数值变化，进入下一天
```

## 三、数值系统

### 3.1 基础数值（游戏原有）

| 数值 | 作用 |
|------|------|
| **人气值** | 粉丝数量，决定直播收益 |
| **体力值** | 每天直播消耗，为0时强制休息 |
| **善良值** | 道德选择，影响NPC好感 |
| **诚信值** | 是否真诚，影响粉丝信任 |
| **经济值** | 金钱，可负（欠债） |

### 3.2 新增数值

| 数值 | 作用 | 影响 |
|------|------|------|
| **精神值** | 理智程度 | 影响直播状态，过低会失控 |
| **人设完整度** | 虚拟vs真实 | 影响结局走向 |

**精神状态等级**：
- 80-100：正常（直播稳定，但弹幕觉得"无聊"）
- 60-79：紧张（偶尔失误，弹幕觉得"有点意思"）
- 40-59：上头（放飞自我，人气暴涨但不可控）
- 20-39：失控（弹幕疯狂，热搜频出）
- 0-19：崩溃（直播间变奇观）

### 3.3 NPC好感度系统

| NPC | 初始好感 | 首次出现 | 好感度影响 |
|-----|---------|----------|-----------|
| **房东太太** | 30 | 第1天 | ≥60时网暴会帮助；0时房租翻倍 |
| **可心** | 80 | 第3天 | 影响精神状态；离世后若被利用善良值-20 |
| **豆豆** | 0 | 第6天（捡到后20） | 影响精神慰藉和人气 |
| **猥琐男** | 10 | 第11天 | ≥20时骚扰降低；0时不再骚扰 |
| **月牙儿** | 20 | 第16天 | ≥50时成为朋友；0时敌对 |
| **妈妈** | 40 | 回忆/电话 | ≥60时会来帮忙；影响最终抉择 |

**好感度提升方式**：
- 在剧情节点做出正确选择
- 日常事件中选择帮助/关心NPC
- 直播中提及NPC（正面）

## 四、事件系统（双轨制）

### 4.1 核心剧情节点（6个）- 必然触发

这6个事件是**主线剧情**，在固定天数必然触发：

#### 节点1：房东太太晕倒（第3-5天随机一天触发）

**前置条件**：已见过房东太太

**剧情**：
某天傍晚，你在楼道撞见房东太太突然晕倒，脸色苍白。同时，电脑提醒今晚8点有重要PK，赢了能获得大额流量扶持和现金奖励。

**选项与结果**：
1. **放弃PK，送医院**（善良值+20，房东太太好感+30）
   - 垫付医药费，全程照顾
   - 后续：房东太太会帮你化解网暴，减免房租

2. **拨打120后继续直播**（人气值+5000，房东太太好感+10）
   - 赢了PK，但房东太太留下永久性后遗症
   - 后续：她儿子接她去外地，无法在你被网暴时帮忙

3. **假装没看见**（善良值-30，房东太太好感归零）
   - 房东太太因救助不及时去世
   - 后续：新房东（她儿子）房租翻倍，限期3天搬离

4. **联系房东儿子后再直播**（人气值+3000，诚信值-10）
   - 房东儿子及时赶到，没有后遗症
   - 后续：房租维持原价，但对你多了几分疏离

#### 节点2：邻居骚扰与网暴（第11-13天触发）

**前置条件**：粉丝量突破1万

**剧情**：
合租的猥琐男见你直播有起色，心生嫉妒，频繁骚扰你，还在网上散布不实信息："小爱私生活混乱，靠不正当关系涨粉"。网暴席卷而来。

**选项与结果**：
1. **自己发声明解释**（诚信值+10，人气值-20）
   - 熬夜写长文澄清，但网友不买账
   - 谣言愈演愈烈，品牌方取消合作

2. **冷处理**（诚信值-15，精神值-20）
   - 不回应，专注直播
   - 短期内谩骂不断，快要大火时谣言被翻出，引发更严重危机

3. **找男邻居对质**（体力值-10，善良值-5）
   - 邻居拒绝还威胁你
   - 若房东太太好感≥60且未去世：她会出面骂走邻居，网暴平息，涨粉更快

4. **寻求平台帮助**（诚信值+5，人气值+30）
   - 提交证据，平台删除不实信息，封禁邻居账号
   - 发布官方声明为你澄清，粉丝量不降反升

#### 节点3：捡到豆豆（第6天触发）

**剧情**：
你在路边发现一只受伤的流浪狗（豆豆），后腿有旧伤。把它捡回家偷偷照顾。

**选项与结果**：
1. **不让豆豆直播，专心照顾**（善良值+20，豆豆好感+30）
   - 拒绝宠物品牌代言
   - 后续：豆豆康复，成为精神支柱，粉丝称赞你有爱心

2. **让豆豆频繁直播赚钱**（经济值+50000，善良值-30）
   - 豆豆状态越来越差，最终因过度劳累去世
   - 你赚了很多钱，但内心充满愧疚

3. **控制直播时长，兼顾**（善良值+10，经济值+20000）
   - 每天只让豆豆出镜10-15分钟
   - 后续：豆豆康复，既获得收益，也保留陪伴

4. **转让代言，委托专业照顾**（善良值+15，经济值+10000）
   - 代言机会转让给其他流浪动物
   - 豆豆在专业照顾下痊愈，口碑良好

#### 节点4：可心的悲剧（第10-12天触发）

**前置条件**：可心好感度>50

**剧情**：
可心兼职越来越多，每天只睡3小时。某天，她在工厂被机器砸伤，心老板拒绝赔偿。你赶到医院，她终于崩溃哭诉自己患有严重抑郁症。几天后，可心从医院天台一跃而下。

**选项与结果**：
1. **利用悲剧"吃人血馒头"**（善良值-50，人气值暴涨）
   - 直播声泪俱下讲述可心故事，晒出她的日记和搜索记录
   - 短期内粉丝暴涨，但内心备受煎熬
   - 最终因"消费逝者"被全网网暴，账号封禁

2. **默默帮助可心家人**（善良值+30，诚信值+20）
   - 用直播收益帮她还清债务，支付妹妹学费和妈妈医药费
   - 每年忌日停播一天去墓前陪伴
   - 粉丝得知后更加尊重你

3. **寻求公众帮助，传递善意**（善良值+25，人气值+20）
   - 发起公益倡议，呼吁关注打工人和抑郁症患者
   - 不刻意卖惨，传递正向价值观

4. **暂停直播，陪伴家人**（善良值+35，人气值-10）
   - 停播一个月，搬到可心家附近陪伴她妈妈和妹妹
   - 复播后粉丝因你的重情重义更加支持你

#### 节点5：与月牙儿的世纪PK（第16-18天触发）

**前置条件**：粉丝量突破50万

**剧情**：
你和另一位主播月牙儿并称为"直播双花"。平台举办大型PK活动，冠军可获得百万奖金和全网推广。PK前一天，你收到匿名邮件，里面是月牙儿的黑料：她打压新人、盗取内容、偷税漏税。

**选项与结果**：
1. **发布黑料，引导舆论**（善良值-40，人气值+100000）
   - 月牙儿被全网网暴，口碑崩塌
   - 你轻松获胜，但内心厌倦这个功利的世界

2. **不发布黑料，公平竞争**（善良值+20，诚信值+20）
   - 删掉黑料，专注打磨内容
   - 月牙儿后来主动找你，两人成为朋友

3. **私下沟通，劝她澄清**（善良值+30，月牙儿好感+40）
   - 联系月牙儿，劝她主动道歉整改
   - 她最终公开道歉，两人公平PK，后续联动双赢

4. **举报给平台处理**（善良值+15）
   - 不公开黑料，交给平台官方处理
   - 月牙儿被处罚，但会记恨你，后续可能散布你的黑料

#### 节点6：母亲的阿尔茨海默症（第20天触发）

**剧情**：
你成为百万粉大博主，事业蒸蒸日上。老家亲戚突然来电：妈妈确诊阿尔茨海默症，已经记不清人了。你手中还有即将签约的顶级品牌合作和无数粉丝期待。

**选项与结果**：
1. **放弃直播，回老家陪伴**（善良值+40，妈妈好感+50）
   - 推掉所有合作，注销账号，全职陪伴
   - 结局：失去主播光环，但收获内心安宁，妈妈偶尔能认出你

2. **请专人照顾，继续直播**（经济值+200000，善良值-20）
   - 请护工24小时照顾，每月抽3天回家
   - 结局：保住事业，但错过陪伴最后时光，余生遗憾

3. **调整节奏，兼顾**（善良值+25，经济值+100000）
   - 推掉部分合作，每天直播2小时，其余时间陪伴
   - 把妈妈接到身边，请住家护工
   - 结局：事业亲情平衡，妈妈病情稳定

4. **带着母亲直播**（善良值+30，人气值+50000）
   - 调整内容，直播照顾母亲的日常
   - 不刻意煽情，但打动无数人
   - 结局：陪伴与事业同行，妈妈看到你的直播会下意识微笑

### 4.2 随机翻车事件池（30个）

**触发机制**：
- 每天晚上直播时，有30%概率触发随机事件
- 若当天有核心剧情节点，则优先触发节点
- 事件类型根据当前天数和分区决定

**事件分类**：

#### 技术翻车类（6个）
- 美颜失效、忘关麦、软件崩溃、弹窗入侵、设备连环故障、AI换脸

#### 社交翻车类（6个）
- 房东乱入、父母查岗、室友乱入、前任出现、被粉丝认出、（猥琐男骚扰已放入节点2）

#### 技能翻车类（6个）
- 游戏连败、才艺翻车、PK惨败、被大佬查房、挑战失败、直播睡觉

#### 人设翻车类（6个）
- 人设崩塌、炫富翻车、学历造假被发现、数据造假被发现、抄袭被揭、剧本PK被揭穿

#### 平台/超现实类（6个）
- MCN签约、平台算法显灵、观众是AI、神秘礼物、直播间闹鬼、穿越

**每个事件的选项结构**：
- 4个选项，每个影响不同数值
- 部分选项会影响NPC好感度
- 部分选项会触发后续连锁事件

## 七、结局系统（5个）

### 结局判定方式

20天结束后，根据以下因素综合判定：
- 善良值、诚信值、精神值、人设完整度
- 6个NPC好感度
- 6个核心剧情节点的选择

### 5个结局

#### 结局一：自我和解·回归本真（偏圆满）

**条件**：
- 善良值≥60，诚信值≥60
- 妈妈、豆豆好感度≥60，可心好感度≥60（或已离世但未被利用）
- 人设完整度>50

**剧情**：
暂停直播，回老家陪伴妈妈，关掉虚拟形象。粉丝反而更喜欢真实的你。妈妈病情稳定，豆豆陪伴，月牙儿成为朋友。你终于明白：互联网身份只是一部分，真正的身份认同是接纳现实中的自己。

#### 结局二：困于虚拟·彻底迷失（偏悲剧）

**条件**：
- 善良值≤40，诚信值≤40
- 妈妈、可心、豆豆好感度≤30
- 人设完整度<30

**剧情**：
成为顶级大博主，拥有千万粉丝和巨额财富，但失去所有现实牵挂——妈妈彻底失忆，可心、豆豆淡出生活，月牙儿敌对。深夜坐在豪华公寓里倍感孤独，最终全网断联，活成了互联网的傀儡。

#### 结局三：逃离网络·回归现实（中性）

**条件**：
- 精神值<20，主动选择退网
- 或遭遇严重网暴后选择放弃

**剧情**：
意识到网络光鲜都是虚无，现实的温暖才是真的。停播100天里彻底告别主播身份，删除账号，回老家找稳定工作，陪伴家人，回归平淡却真实的现实生活。

#### 结局四：双向崩塌·彻底沉寂（失败）

**条件**：
- 经济值≤-1000
- 房东太太好感度=0（或已去世且新房东驱逐）
- 善良值<20

**剧情**：
被赶出出租屋，经济破产，设备被变卖，人气归零。所有人物淡出生活，无法继续直播，也无力回家面对妈妈，重新沦为无人问津的普通人。

#### 结局五：真相大白·双向救赎（完美）

**条件**：
- 善良值≥70，诚信值≥70
- 所有NPC好感度≥60
- 在节点6选择"带着母亲直播"或"调整节奏兼顾"

**剧情**：

## 九、总结

### 核心设计亮点

1. **双轨事件系统**：6个主线节点 + 30个随机事件，既有剧情又有重玩性
2. **NPC好感度融入日常**：不是独立系统，而是通过选择和事件自然提升
3. **元叙事结构**：玩家作为粉丝穿越，最终理解停播真相
4. **5个结局**：根据20天选择综合判定，有明确触发条件
5. **弹幕人格**：8个有记忆的粉丝，让AI弹幕更有趣
6. **与现有系统融合**：天赋树、四分区、基础数值都不冲突

### 一句话总结

> 让玩家笑着笑着发现"卧槽这不就是我"，最终理解：身份认同不是选择虚拟或现实，而是在两者之间找到真正的自己。
