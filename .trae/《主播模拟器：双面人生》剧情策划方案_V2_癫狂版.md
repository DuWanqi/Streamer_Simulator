# 《主播模拟器：双面人生》剧情策划方案 V2 - 癫狂版

> **核心理念**：不是来教育玩家的，是来让玩家爽、让玩家上头、让玩家笑到肚子疼，最后笑着笑着哭出来的。

---

## 一、核心玩法重构

### 1.1 全新数值系统 - "精神状态仪表盘"

玩家需要同时管理5个核心数值，所有数值都可视化显示（进度条+emoji）：

| 数值 | 图标 | 作用 | 过高后果 | 过低后果 |
|------|------|------|----------|----------|
| **粉丝数** | 👥 | 基础目标，要涨粉 | 黑粉比例增加 | 游戏失败 |
| **精神值** | 🧠 | 主角的理智程度 | 触发"超天酱"模式（见下文） | 抑郁结局 |
| **道德底线** | ⚖️ | 还能不能算个人 | 解锁无下限操作 | 圣母结局 |
| **钱包厚度** | 💰 | 能不能活下去 | 被MCN绑架 | 饿死 |
| **人设完整度** | 🎭 | 虚拟vs真实的撕裂感 | 彻底迷失 | 暴露真实自我 |

**关键机制**：
- 任何操作都会影响多个数值（trade-off设计）
- 数值之间会相互转化：比如"精神值↓"可以换来"粉丝数↑"
- 每天结束时，根据数值组合触发随机事件

### 1.2 "超天酱"模式 - 核心爽点

当**精神值>80**时，小爱进入"超天酱"状态：
- 直播效果爆炸，所有收益×3
- 解锁疯狂操作：直播睡觉、直播发呆、直播骂粉丝（反向涨粉）
- 弹幕变成"主播疯了但好可爱"
- 但精神值会快速下降，降到0会触发"崩溃直播"

**崩溃直播**：
- 小爱在直播间大哭/大笑/胡言乱语
- 粉丝数暴涨（吃瓜观众涌入）
- 但第二天会触发"网暴事件"

### 1.3 直播流派系统 - Roguelike Build

每场直播可以选择一个"流派"，不同流派影响数值变化：

| 流派 | 核心机制 | 适合玩家 |
|------|----------|----------|
| **炒作狗** | 道德底线↓=粉丝数↑↑ | 喜欢无下限的玩家 |
| **实力派** | 才艺↑=稳定涨粉 | 稳健型玩家 |
| **抽象派** | 随机事件概率↑ | 喜欢刺激的玩家 |
| **卖惨流** | 精神值↓=同情粉↑ | 情感操控大师 |
| **技术宅** | 知识↑=高质量粉↑ | 硬核玩家 |
| **颜值党** | 外貌↑=土豪打赏↑ | 简单直接 |

**Build组合示例**：
- "疯批美人"：炒作狗+卖惨流+超天酱模式
- "知识抽象家"：技术宅+抽象派
- "纯欲天花板"：颜值党+实力派

## 二、主角设定重构 - "小爱"

### 2.1 基础人设

**姓名**：小爱（AI，暗示她是"被算法选中的人"）
**年龄**：22岁
**身份**：刚毕业大学生，找不到工作，被迫当主播
**口头禅**：
- "家人们谁懂啊"
- "这破直播是一天也播不下去了"
- "谢谢大哥的火箭，大哥大气"
**隐藏设定**：
- 有严重的双重人格倾向（玩家选择会强化某一面）
- 对互联网文化了如指掌，但内心鄙视这一切
- 有一个"真实自我"计数器，太高会触发"不想演了"事件

### 2.2 小爱的心理状态可视化

在屏幕角落显示小爱的心理状态emoji：
- 😇 天使模式：正常直播，人畜无害
- 😈 恶魔模式：想搞事，想炒作
- 🤡 小丑模式：精神值低，开始发疯
- 💀 虚无模式：彻底迷失，不知道自己在干嘛
- 🌟 超天酱模式：精神值爆表，进入癫狂状态

## 三、随机事件池（25个核心事件）

每场直播后会触发1-2个随机事件，玩家需要做出选择。

### 3.1 无下限炒作类（5个）

**事件1：前任出现**
- 描述：前任突然出现在直播间刷礼物，弹幕开始吃瓜
- 选项：
  1. 直播连麦撕逼（道德底线-20，粉丝数+5000，精神值-10）
  2. 假装不认识（道德底线+5，无事发生）
  3. 复合炒作（道德底线-30，粉丝数+10000，解锁"假情侣"剧情线）
  4. 让现任来对线（道德底线-50，粉丝数+20000，精神值-20，触发"修罗场"事件）

**事件2：假死**
- 描述：你发了一条"最后的直播"动态，粉丝以为你要自杀
- 选项：
  1. 真的装死（道德底线-100，粉丝数+50000，第二天"复活"被骂上热搜）
  2. 澄清是误会（粉丝数-1000，被说"炒作狗"）
  3. 顺势卖惨（道德底线-30，粉丝数+10000，精神值-20）
  4. 真去死（游戏结束，解锁结局"互联网殉道者"）

**事件3：蹭热度**
- 描述：某明星塌房了，全网都在吃瓜
- 选项：
  1. 开直播锐评（粉丝数+3000，可能被明星粉丝冲）
  2. 假装是圈内人爆料（道德底线-20，粉丝数+8000，可能被起诉）
  3. 唱一首《凉凉》（抽象派+1，粉丝数+1000）
  4. 装死不理（无事发生）

**事件4：剧本PK**
- 描述：另一个主播约你PK，但提议先对剧本
- 选项：
  1. 同意剧本（道德底线-10，稳定收益）
  2. 假戏真做，中途背刺（道德底线-40，粉丝数+15000，被全网骂）
  3. 拒绝，真PK（50%胜率）
  4. 举报对方（无事发生，但对方会记仇）

**事件5：MCN签约**
- 描述：MCN机构找上门，给了一份"卖身契"
- 选项：
  1. 签约（钱包+100000，每天必须播8小时，精神值每天-5）
  2. 拒绝（无事发生）
  3. 假装签约然后跑路（道德底线-50，被全网封杀）
  4. 曝光合同（道德底线+20，被MCN追杀）

## 六、与现有系统的适配

### 6.1 数值系统扩展

在现有`PlayerState`基础上新增：
```typescript
interface PlayerState {
  // 原有数值
  followers: number;
  fanClub: number;
  income: number;
  // ...其他原有数值
  
  // 新增数值
  sanity: number;           // 精神值 0-100
  morality: number;         // 道德底线 0-100
  personaIntegrity: number; // 人设完整度 0-100
  
  // 流派等级
  streamStyle: {
    hypeDog: number;        // 炒作狗
    skilled: number;        // 实力派
    abstract: number;       // 抽象派
    pitySeeker: number;     // 卖惨流
    techGeek: number;       // 技术宅
    faceValue: number;      // 颜值党
  };
  
  // 隐藏数值
  trueSelf: number;         // 真实自我暴露度
  aiLove: number;           // 与AI观众的好感度
  superAngelMode: number;   // 超天酱模式触发次数
}
```

### 6.2 事件系统扩展

`EventPool`扩展为25个事件，每个事件有：
- 触发条件（数值阈值、前置事件等）
- 多个选项，每个选项影响不同数值
- 随机结果（部分选项有概率分支）

### 6.3 结局判定系统

结局判定采用"标签系统"：
- 每个选择会给玩家打上标签
- 最终根据标签组合决定结局
- 支持多条件触发（如"疯批+无下限"=疯批美人）

## 七、核心体验循环

```
开始直播 -> 选择流派 -> 进行直播（AI弹幕互动）-> 触发随机事件 -> 做出选择 -> 数值变化 -> 
进入超天酱/崩溃模式？-> 下播 -> 看数据复盘 -> 升级属性 -> 下一天
```

**爽点设计**：
1. **即时反馈**：每个选择立即看到数值变化
2. **build乐趣**：不同流派组合有不同玩法
3. **意外惊喜**：随机事件带来不可预测性
4. **梗文化**：玩家能get到笑点，产生共鸣
5. **多周目**：25个结局驱动重复游玩

## 八、总结

### 这个版本的优势：

1. **游戏性强**：数值系统+随机事件+build组合，有重玩价值
2. **传播性好**：梗多、笑点密、截图价值高
3. **上头**：超天酱模式、数值成长、多结局收集
4. **有深度**：笑着笑着会发现"这不就是我/我关注的主播吗"
5. **适配AI**：AI生成弹幕可以融入大量网络梗

### 与原版的区别：

| 维度 | 原版 | 新版 |
|------|------|------|
| 核心体验 | 读故事 | 玩游戏 |
| 玩家动机 | 体验剧情 | 爽、上头、收集 |
| 选择设计 | 道德抉择 | 数值trade-off |
| 结局数量 | 5个 | 25个 |
| 事件数量 | 6个固定 | 25个随机 |
| 主题表达 | 严肃深刻 | 黑色幽默 |
| 传播点 | 感人故事 | 梗、笑点、猎奇 |

---

**一句话总结**：
> 不是来教育玩家的，是来让玩家笑着笑着发现"卧槽这不就是我"的。
