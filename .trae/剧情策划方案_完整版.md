# 《主播模拟器：双面人生》剧情策划方案 - 完整版

> **核心理念**：翻车即内容，失控即流量，社死即封神。
> 
> **精神内核**：没有说教，只有讽刺 — 社会批判藏在笑声里。

---

## 一、核心设定

### 1.1 故事框架（元叙事）

**开篇**：
- 玩家是小爱的忠实粉丝，停播100天后收到直播推送
- 观看时突然穿越，绑定角色林小爱，回到起步阶段（第1天）
- 机械音："请走完小爱的人生之路，找到她停播100天的真相"

**真相**（20天后揭示）：
> 小爱并非遭遇意外，而是在虚拟与现实两种身份的挣扎中陷入精神内耗，选择暂停直播，寻找真实自我。

### 1.2 小爱是谁

- **姓名**：林小爱（主播名：小爱）
- **年龄**：22岁
- **身份**：刚毕业大学生，找不到工作，被迫当主播还债
- **双重身份**：
  - 镜头前：闪闪发光的小太阳，被人喜欢、充满自信
  - 镜头后：家境普通、生活窘迫、被家人否定的底层年轻人

## 二、游戏流程（20天结构）

### 2.1 整体时间线

| 天数 | 阶段 | 核心内容 | 出现NPC |
|------|------|----------|---------|
| 第1天 | 开篇 | 玩家穿越，初始探索 | 房东太太（首次） |
| 第2-5天 | 初期生存 | 房租危机，努力涨粉 | 可心（第3天出现） |
| 第6-10天 | 涨粉阶段 | 捡到豆豆，粉丝增长 | 豆豆（第6天出现） |
| 第11-15天 | 网暴危机 | 猥琐男骚扰，应对网暴 | 猥琐男（第11天出现） |
| 第16-19天 | 巅峰与迷失 | 与月牙儿PK，成为大主播 | 月牙儿（第16天出现） |
| 第20天 | 最终抉择 | 母亲阿尔茨海默症 | 妈妈（电话/回忆） |
| 结局 | 真相揭示 | 停播100天真相 + 5种结局 | 所有NPC |

### 2.2 每日流程

```
每天固定流程：
1. 白天：现实事件（可能触发NPC互动）
   - 检查生存状态（房租、水电、食物）
   - 可能触发现实支付事件
2. 晚上：开启直播
   - AI弹幕互动（8人格系统）
   - 可能触发：随机翻车事件 或 核心剧情节点
   - 玩家做出选择
   - 数值变化
3. 下播：热搜生成（如有重大事件）
4. 结算：查看数值变化，进入下一天
```

## 三、数值系统

### 3.1 基础数值

| 数值 | 作用 |
|------|------|
| **人气值** | 粉丝数量，决定直播收益 |
| **体力值** | 每天直播消耗，为0时强制休息 |
| **善良值** | 道德选择，影响NPC好感 |
| **诚信值** | 是否真诚，影响粉丝信任 |
| **经济值** | 金钱，可负（欠债） |
| **精神值** | 理智程度，影响直播状态 |
| **人设完整度** | 虚拟vs真实，影响结局 |

### 3.2 NPC好感度系统

| NPC | 初始好感 | 首次出现 | 好感度影响 |
|-----|---------|----------|-----------|
| **房东太太** | 30 | 第1天 | ≥60时网暴会帮助；0时房租翻倍 |
| **可心** | 80 | 第3天 | 影响精神状态；离世后若被利用善良值-20 |
| **豆豆** | 0 | 第6天（捡到后20） | 影响精神慰藉和人气 |
| **猥琐男** | 10 | 第11天 | ≥20时骚扰降低；0时不再骚扰 |
| **月牙儿** | 20 | 第16天 | ≥50时成为朋友；0时敌对 |
| **妈妈** | 40 | 回忆/电话 | ≥60时会来帮忙；影响最终抉择 |

## 四、现实生存支付系统

### 4.1 每日固定支出

| 项目 | 每日费用 | 说明 |
|------|---------|------|
| **房租** | 100/天 | 拖欠7天后房东催缴，拖欠10天被驱逐 |
| **水电费** | 20/天 | 拖欠3天断水断电，无法直播 |
| **食物** | 30/天 | 体力值恢复必需，不吃体力值-20/天 |
| **网费** | 10/天 | 拖欠1天断网，无法直播 |

**每日总支出：160**

### 4.2 收入方式

| 方式 | 收益 | 条件 |
|------|------|------|
| **直播打赏** | 人气值×0.1~0.5 | 正常直播 |
| **品牌合作** | 5000~50000 | 人气值>1万后随机触发 |
| **带货分成** | 销售额×10% | 解锁带货技能后 |
| **兼职打工** | 100/天 | 放弃直播时间去打工 |

### 4.3 生存危机事件

**房租危机**（第3-5天必然触发）：
- 拖欠7天，房东下达最后通牒
- 选项：
  1. 外出打工赚取（体力值-30，经济值+300，人气值-1000）
  2. 恳求延期（需房东好感≥40，或诚信值>60）
  3. 直播借钱（人气值+2000，诚信值-10，可能收到打赏）
  4. 假装不在家（房东断网，直播中断1天）

**断水断电**：
- 水电费拖欠3天触发
- 无法直播，只能外出或休息
- 恢复后需要支付滞纳金（欠费×1.5）

**饥饿状态**：
- 连续2天不吃食物触发
- 体力值每天-20
- 直播时可能晕倒（触发特殊事件）

## 五、事件系统（双轨制）

**剧情**：可心工伤后被辞退，抑郁症爆发，最终跳楼。

## 九、结局系统（5个）

### 9.1 结局判定

根据以下因素综合判定：
- 善良值、诚信值、精神值、人设完整度
- 6个NPC好感度
- 6个核心剧情节点选择

### 9.2 5个结局

#### 结局一：自我和解·回归本真（偏圆满）

**条件**：
- 善良值≥60，诚信值≥60
- 妈妈、豆豆好感度≥60，可心好感度≥60（或已离世但未被利用）
- 人设完整度>50

**剧情**：
暂停直播，回老家陪伴妈妈，关掉虚拟形象。粉丝反而更喜欢真实的你。妈妈病情稳定，豆豆陪伴，月牙儿成为朋友。你终于明白：互联网身份只是一部分，真正的身份认同是接纳现实中的自己。

#### 结局二：困于虚拟·彻底迷失（偏悲剧）

**条件**：
- 善良值≤40，诚信值≤40
- 妈妈、可心、豆豆好感度≤30
- 人设完整度<30

**剧情**：
成为顶级大博主，拥有千万粉丝和巨额财富，但失去所有现实牵挂——妈妈彻底失忆，可心、豆豆淡出生活，月牙儿敌对。深夜坐在豪华公寓里倍感孤独，最终全网断联，活成了互联网的傀儡。

#### 结局三：逃离网络·回归现实（中性）

**条件**：
- 精神值<20，主动选择退网
- 或遭遇严重网暴后选择放弃

**剧情**：
意识到网络光鲜都是虚无，现实的温暖才是真的。停播100天里彻底告别主播身份，删除账号，回老家找稳定工作，陪伴家人，回归平淡却真实的现实生活。

#### 结局四：双向崩塌·彻底沉寂（失败）

**条件**：
- 经济值≤-1000
- 房东太太好感度=0（或已去世且新房东驱逐）
- 善良值<20

**剧情**：
被赶出出租屋，经济破产，设备被变卖，人气归零。所有人物淡出生活，无法继续直播，也无力回家面对妈妈，重新沦为无人问津的普通人。

#### 结局五：真相大白·双向救赎（完美）

**条件**：
- 善良值≥70，诚信值≥70
- 所有NPC好感度≥60
- 在节点6选择"带着母亲直播"或"调整节奏兼顾"

**剧情**：
妥善处理所有矛盾，用真诚收获现实认可与网络喜爱。停播100天真相揭晓：你并非遭遇意外，而是在两种身份的挣扎中选择暂停，寻找真实自我。最终带着粉丝理解和现实温暖回归，用真实自我直播，实现双向救赎。

## 十二、总结

### 核心设计亮点

1. **现实生存系统**：房租、水电、食物，增加真实感
2. **双轨事件系统**：6个主线节点 + 30个随机事件

4. **弹幕人格系统**：8个有记忆的粉丝
5. **热搜系统**：AI生成，增加传播点
6. **AI生涯总结**：个性化纪录片
7. **元叙事结构**：玩家作为粉丝穿越，理解停播真相
8. **5个结局**：根据20天选择综合判定
