# 电脑桌面APP系统 Spec

## Why

当前游戏的主界面（房间）中间的电脑屏幕只是一个静态图片，没有交互功能。用户需要一个可以点击的电脑屏幕，进入电脑桌面后可以使用微信和微博两个APP：
- **微信APP**：与NPC聊天，交房租水电费
- **微博APP**：发帖和查看热搜，与游戏的热搜系统结合

## What Changes

### 新增内容

1. **电脑屏幕交互**
   - 在主界面（main_hub）的电脑屏幕上添加点击交互
   - 点击后进入电脑桌面界面

2. **电脑桌面界面**
   - 模拟Windows/Mac桌面风格
   - 桌面上显示APP图标：微信、微博
   - 底部有任务栏
   - 可以关闭桌面返回房间

3. **微信APP界面**
   - 左侧聊天列表（房东太太、可心、妈妈等NPC）
   - 右侧聊天窗口
   - 支持发送消息和接收NPC回复
   - 房东太太聊天中可以交房租和水电费
   - 聊天内容根据游戏进度和NPC关系动态变化

4. **微博APP界面**
   - 顶部导航栏：首页、热搜、我的
   - 首页：显示主角发的帖子
   - 热搜页：显示游戏中的热搜列表，与HotSearchSystem集成
   - 可以发布新帖子

## Impact

- **Affected files**:
  - `src/game/Game.ts` - 添加电脑桌面和APP渲染方法
  - `src/styles/computer-apps.css` - APP样式
  - `src/systems/NPCSystem.ts` - 可能需要扩展NPC对话数据
  - `src/systems/HotSearchSystem.ts` - 集成到微博APP

## ADDED Requirements

### Requirement: 电脑屏幕交互

**实现要求**：
- 在main_hub的电脑屏幕上添加点击事件
- 点击后显示电脑桌面界面（全屏覆盖）
- 桌面背景使用深色主题（类似Windows 11）

### Requirement: 电脑桌面界面

**布局**：
```
┌─────────────────────────────────────────┐
│  桌面背景（深色渐变）                     │
│                                         │
│   [微信图标]   [微博图标]                 │
│                                         │
│                                         │
│  ─────────────────────────────────────  │
│  任务栏 [开始] [微信] [微博] [时间]      │
└─────────────────────────────────────────┘
```

**功能**：
- APP图标双击打开对应APP
- 任务栏显示已打开的APP
- 点击关闭按钮返回房间界面

### Requirement: 微信APP界面

**布局**：

```
┌─────────────────────────────────────────┐

│  聊天列表         │  聊天窗口             │

│  │ 👵 房东太太 │  │  │ 房东太太        │  │
│  │ 房租记得交哦│  │  ├────────────────┤  │
│  └────────────┘  │  │                │  │
│  ┌────────────┐  │  │ [对方消息]      │  │
│  │ 👩 可心     │  │  │ [我的消息]      │  │
│  │ 在吗？      │  │  │                │  │
│  └────────────┘  │  │ [交房租按钮]    │  │

└──────────────────┴──────────────────────┘
```

**功能**：
- 左侧显示NPC列表，带未读消息提示
- 点击NPC打开对应聊天窗口
- 聊天窗口显示历史消息
- 特定NPC（房东太太）显示操作按钮（交房租、交水电）
- 可以发送消息，NPC根据关系值回复

**NPC聊天数据**：
- 房东太太：催租、收租确认、日常问候
- 可心：主播交流、情感支持
- 妈妈：关心身体、寄钱
- 豆豆：狗狗表情（收养后）

## UI风格要求

- 参考用户提供的微信截图（深色主题，绿色气泡）
- 参考用户提供的微博截图（白色卡片，橙色强调）
- 使用CSS变量方便后续替换美术资源
- 响应式设计，适配不同屏幕尺寸
