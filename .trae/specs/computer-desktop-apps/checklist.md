# 电脑桌面APP系统 Checklist

## 电脑桌面功能

- [ ] 电脑屏幕可以点击
- [ ] 点击后进入电脑桌面界面
- [ ] 桌面显示微信和微博APP图标
- [ ] 底部有任务栏
- [ ] 可以关闭桌面返回房间

## 微信APP功能

- [ ] 微信APP窗口可以打开
- [ ] 左侧显示NPC聊天列表
- [ ] 点击NPC可以打开对应聊天窗口
- [ ] 聊天窗口显示历史消息
- [ ] 可以发送消息
- [ ] 房东太太聊天显示交房租按钮
- [ ] 房东太太聊天显示交水电按钮
- [ ] 点击交租按钮可以支付

## 微博APP功能

- [ ] 微博APP窗口可以打开
- [ ] 顶部有导航栏（首页、热搜、我的）
- [ ] 首页显示主角发布的帖子
- [ ] 热搜页显示游戏中的热搜列表
- [ ] 热搜数据与HotSearchSystem同步
- [ ] 可以发布新帖子
- [ ] 发布帖子影响粉丝数

## UI样式

- [ ] 微信使用深色主题
- [ ] 微信消息气泡：自己绿色，对方白色
- [ ] 微博使用白色卡片风格
- [ ] 热搜使用橙色强调色
- [ ] 整体风格统一美观

## 集成测试

- [ ] 从房间界面可以打开电脑
- [ ] 在微信中可以交房租
- [ ] 在微信中可以与NPC聊天
- [ ] 在微博中可以查看热搜
- [ ] 在微博中可以发布帖子
- [ ] 关闭APP后返回桌面
- [ ] 关闭桌面后返回房间
